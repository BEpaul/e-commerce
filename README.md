## Introduction
상품 주문 및 결제 기능을 메인으로 하는 이커머스 서비스 프로젝트입니다. 

## Getting Started

### Prerequisites

#### Running Docker Containers

`local` profile 로 실행하기 위하여 인프라가 설정되어 있는 Docker 컨테이너를 실행해주셔야 합니다.

```bash
docker-compose up -d
```
